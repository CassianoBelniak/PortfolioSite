Cores de Preenchimento

O último recurso da área de formatação a ser estudado neste curso são as cores de preenchimento. Utilizando cores de preenchimento combinado com cores de fontes pode-se atingir um resultado muito agradável na apresentação de uma planilha.

Aplicar uma cor de preenchimento segue a mesma mecânica para se aplicar cores às fontes. É necessário selecionar o intervalo onde se deseja aplicar as cores de fundo e clicar no botão

Para ter acesso à uma paleta de cores é só clicar na seta ao lado do botão cor da fonte e selecionar a cor desejada.
