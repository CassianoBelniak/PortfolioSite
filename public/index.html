ISaw

A LGPD (Lei Geral de Proteção de Dados Pessoais) é a nova lei brasileira de proteção de dados baseada na lei européia GDPRque passa a valer em 2021.

Para o Global Legal Hackathon, nós desenvolvemos uma solução que permite verificar se o seu site se enquadra nas novas diretrizes da lei brasileira de proteção de dados.

O site foi desenvolvido utilizando Node, Pug, Sass e MySql.

Paraíso

O Paraíso foi um projeto desenvolvido para a faculdade de design. O objetivo era transformar uma foto comum de praia em um cartaz de um filme de suspense.

Precisei aplicar alguns filtros para escurecer a imagem, remover algumas sombras, adicionar um novo fundo para substituir o céu limpo e adicionar um texto com o nome do suposto filme. Era necessário o uso da imagem de praia disponível pelo professor.

Montagem realizada no Krita.

Robô tridimensional

Outro trabalho desenvolvido para a faculdade. Nessa atividade era necessário desenhar uma figura utilizando um ponto de fuga e colori-lo usando cores frias.

Desenho feito a mão, imagem escaneada melhorada com o Krita e colorido no Inkscape.

One Hour Game Jam

O One Hour Game Jam é um desafio semanal em que você precisa desenvolver um jogo completo (programação, arte, musica) em apenas uma hora.

Os jogos devem ser seguir um tema que é anunciado na hora da competição. A comunidade faz uma transmição ao vivo dos jogos sendo testados no final de cada sessão.

Não há um ranking por não ser uma competição.

A maior parte desses jogos foi desenvolvida com o Game Maker
